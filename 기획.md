## 2. 화면 구조 (IA)

```
대시보드
├── 메인 페이지 (평가 개요)
│   ├── 평가 개요 바 (응시자 현황, 평균 스킬 확보율)
│   ├── [탭] 평가 결과
│   │   ├── 점수 분포 (KDE 곡선, 과목별 탭)
│   │   ├── 역량 획득 현황 (도넛 차트)
│   │   └── 평가 총평 (전체 + 트랙별)
│   └── [탭] 응시자 목록
│       └── 필터/정렬/검색 + 테이블 + 페이지네이션
│
└── 평가자 개인 페이지
    ├── 헤더: 이름 / 이메일 / 완료 여부 / 날짜 / 소요시간 / 행동 분석 버튼
    │   └── 행동 분석 패널: 영상 | 타임라인
    ├── [탭] 결과 요약
    │   ├── 총점 / 획득 역량 수 / 순위
    │   ├── 과목별 점수 (Progress Bar)
    │   ├── 점수 분포 (KDE, 내 위치 마커)
    │   └── 시험 결과 총평
    ├── [탭] 클라우드 & 분산 시스템
    ├── [탭] 클라우드 컴퓨팅
    └── [탭] Python
        ※ 트랙 탭 공통 구성:
           - 트랙 설명
           - 트랙 점수 / 등수 / 소요시간 / 획득 역량 수(확보율)
           - 점수 분포 (KDE, 내 위치 마커)
           - 역량별 레이더 (나 vs 전체 평균, 호버 툴팁)
           - 평가 총평
           - 스킬별 세부 결과 (항상 펼침, 장단점 포함)
```

## 3. 사용자 플로우

```
[메인 페이지 진입]
    │
    ├── [평가 결과 탭] 통계/차트 확인
    │
    └── [응시자 목록 탭]
            │
            ├── 필터: 응시 완료 / 미완료
            ├── 정렬: 점수 / 스킬확보율 / 소요시간
            ├── 검색: 이름, 이메일
            │
            └── "결과 보기" 버튼 클릭
                    │
                [평가자 개인 페이지]
                    ├── 헤더 확인 (이메일/완료여부/날짜/소요시간)
                    ├── 행동 분석 버튼 → 영상 / 타임라인
                    │
                    ├── [결과 요약 탭]
                    │   ├── 총점 / 획득 역량 / 순위 카드
                    │   ├── 과목별 점수 바
                    │   ├── 점수 분포 (내 위치 표시)
                    │   └── 시험 결과 총평
                    │
                    └── [트랙별 탭 (3개)]
                        ├── 트랙 설명
                        ├── 점수 / 등수 / 소요시간 / 역량 카드
                        ├── 점수 분포 (내 위치 표시)
                        ├── 레이더 차트 (나 짙은색 / 평균 옅은색, 호버)
                        ├── 평가 총평
                        └── 스킬별 세부 결과 (펼침, 강점/단점)
```

## 4. 화면 상세 기획

### 4.1 메인 페이지 (평가 개요 대시보드)

**페이지 목적:** 전체 평가 현황을 한눈에 파악하고 평가자 목록을 관리

---

#### 4.1.1 평가 개요 바

| 구성 요소 | 내용 |
|----------|------|
| 평가 기간 | 시작일 ~ 종료일, 제한 시간 |
| 응시자 현황 | 완료 N명 (X%) / 전체 N명 |
| 평균 스킬 확보율 | X% |

---

#### 4.1.2 평가 결과 탭

**점수 분포**

| 구성 요소 | 내용 |
|----------|------|
| 탭 버튼 | 전체 평가 결과 / 분산 아키텍처 / 클라우드 컴퓨팅 / Python |
| KDE 곡선 | 매끄러운 밀도 곡선, 불합격(빨강)/합격(초록) 배경 구간 전체 폭 표시 |
| 합격선 | 80점 기준 초록 실선 |
| 평균선 | 주황 점선 |
| 범례 | 그래프 하단에 평균(주황 점선) / 합격선(초록 실선) 범례 표시 |
| 우측 통계 | 평균 / 중앙값 / 범위 / 표준편차 (레이블 → 값 순) |

**역량 획득 현황**

| 구성 요소 | 내용 |
|----------|------|
| 요약 통계 | 전체 획득률 / 인당 평균 획득 / 전체 스킬 수 |
| 도넛 차트 | 전체 + 트랙별 (전체와 트랙 사이 구분선) |

**평가 총평**

| 구성 요소 | 내용 |
|----------|------|
| 전체 총평 | 평균·분포·역량 획득 종합 텍스트 |
| 트랙별 총평 | 각 트랙 이름 + 텍스트 (구분선으로 분리) |

---

#### 4.1.3 응시자 목록 탭

| 컨트롤 | 옵션 |
|--------|------|
| 검색 | 이름, 이메일 |
| 상태 필터 | 전체 / 완료 / 미완료 |
| 정렬 | 점수순 / 스킬확보율순 / 소요시간순 |

**테이블 컬럼:** 응시자 / 상태 / 총점 / 스킬 확보율 / 소요시간 / 결과 보기 버튼

---

### 4.2 평가자 개인 페이지

**페이지 목적:** 특정 평가자의 전체 시험 결과를 상세하게 파악

**진입 경로:** 응시자 목록 탭 > "결과 보기" 버튼 클릭

---

#### 4.2.1 페이지 헤더

| 컴포넌트 | 내용 |
|----------|------|
| 아바타 | 이름 첫 글자 |
| 이름 | 대형 타이틀 |
| 메타 정보 | ✉ 이메일 / 완료 여부 배지 / 📅 날짜 / ⏱ 소요시간 |
| 행동 분석 버튼 | 클릭 시 모달 팝업 |
| 행동 분석 모달 | 왼쪽: 화면 녹화 영상 + 재생 컨트롤 / 오른쪽: 타임라인 (이벤트 목록) |

---

#### 4.2.2 결과 요약 탭

**블록 1 — 평가 결과**

| 구성 요소 | 내용 |
|----------|------|
| 지표 카드 3열 | 총점 (합격 여부) / 순위 / 소요시간 (전체 제한 2:40:00 기준) |
| 트랙별 점수 3열 | 트랙명 + 점수 |

**블록 2 — 점수 분포 & 획득 역량**

| 구성 요소 | 내용 |
|----------|------|
| KDE 곡선 | 내 위치 마커 (콜아웃 박스, 점수 표시), 합격선/평균선 색상 구분 |
| 우측 통계 패널 | 평균 / 중앙값 / 표준편차 |
| 범례 | 그래프 하단 평균(주황 점선) / 합격선(초록 실선) |
| 획득 역량 도넛 | 전체 + 트랙별 (3개) 도넛 차트 |

**시험 결과 총평**

| 구성 요소 | 내용 |
|----------|------|
| 총평 | 종합 텍스트 |

---

#### 4.2.3 트랙별 탭 (클라우드 & 분산 시스템 / 클라우드 컴퓨팅 / Python)

**트랙별 제한시간**

| 트랙 | 제한시간 |
|------|---------|
| 클라우드 & 분산 시스템 아키텍처 | 0:40:00 |
| 클라우드 컴퓨팅 | 0:30:00 |
| Python | 2:00:00 |

**블록 1 — 평가 결과**

| 구성 요소 | 내용 |
|----------|------|
| 지표 카드 4열 | 트랙 점수 / 순위 / 역량 확보 (N/전체개) / 소요시간 (트랙 제한시간 기준) |

**블록 2 — 점수 분포 & 역량별 분석**

| 구성 요소 | 내용 |
|----------|------|
| 점수 분포 | KDE 곡선, 내 위치 마커, 합격선/평균선 색상 구분, 범례, 우측 통계 패널 |
| 역량별 분석 | 나(짙은색 실선) vs 전체 평균(옅은색 점선) 레이더 차트, 축 호버 시 툴팁 |
| 역량 현황 목록 | 스킬명 + 획득/보완/미획득 상태 (레이더 하단) |

**평가 총평 / 스킬별 세부 결과**

| 구성 요소 | 내용 |
|----------|------|
| 평가 총평 | 트랙 총평 텍스트 |
| 스킬별 세부 결과 | 항상 펼침 — 배지 / 스킬명 / 전체 획득률 / 설명 / 강점·보완·단점 판정 |

**레이더 차트 상세**

| 항목 | 내용 |
|------|------|
| 나 (My) | 짙은색 실선 폴리곤 |
| 전체 평균 | 옅은 회색 점선 폴리곤 |
| 호버 | 축 근처 hover 시 "내 점수 X% \| 전체 평균 Y%" 툴팁 표시 |

**스킬 세부 판정 기준**

| 레벨 | 판정 | 배경색 |
|------|------|--------|
| 획득 | 강점 | 초록 |
| 보완 필요 | 보완 | 노랑 |
| 미획득 | 단점 | 빨강 |

## 5. 공통 컴포넌트 명세

| 컴포넌트명 | 설명 | 사용 화면 |
|-----------|------|----------|
| `StatCard` | 단일 수치 강조 카드 (라벨 + 값 + 서브텍스트) | 전 페이지 |
| `StatusBadge` | 상태 표시 배지 (완료/미완료/획득/미획득) | 전 페이지 |
| `ProgressBar` | 점수/비율 시각화 바 | 개인 페이지 |
| `KDECurve` | 커널 밀도 추정 곡선 + 평균/중앙값/개인 마커 | 메인, 개인 |
| `DualRadarChart` | 나 vs 평균 이중 폴리곤 레이더 + 호버 툴팁 | 트랙 탭 |
| `DonutChart` | 비율 도넛 차트 | 메인 페이지 |
| `DataTable` | 필터/정렬/검색 가능한 테이블 | 메인 페이지 |
| `CommentBox` | 텍스트 총평/코멘트 박스 | 전 페이지 |
| `TagGroup` | 역량/강점/약점 태그 그룹 | 개인 페이지 |
| `SkillDetailCard` | 스킬 세부 카드 (항상 펼침, 설명+판정) | 트랙 탭 |
| `BehaviorModal` | 행동 분석 모달 (영상 + 재생 컨트롤 / 타임라인 이벤트 목록) | 개인 페이지 |
