## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 평가 관리 대시보드 |
| 주요 사용자 | 기업 고객 담당자 |
| 목적 | 소속 직원들의 역량 평가 결과를 한눈에 파악하고 개인별 상세 결과를 조회 |
| 주요 기능 | 평가 통계 조회, 평가자 목록 관리, 개인별 결과 조회, 역량별 세부 분석 |

## 2. 화면 구조 (IA)

```
대시보드
├── 메인 페이지 (평가 개요)
│   ├── 상단: 통계 카드 + 시각화 차트
│   └── 하단: 평가자 목록 테이블
│
└── 평가자 개인 페이지
    ├── 상단: 시험 결과 요약
    └── 하단: 평가별 세부 내용 접근 버튼
        └── 평가 세부 내용 페이지
            ├── 상단: 평가 요약 정보
            └── 하단: 역량별 획득 현황 + 통계 그래프
```

---

## 3. 사용자 플로우

```
[메인 페이지 진입]
    │
    ├── 통계/차트 확인
    │
    ├── 필터: 응시 완료 / 미완료
    ├── 정렬: 점수 or 소요시간 / 오름차순 or 내림차순
    │
    └── 평가자 행 클릭
            │
        [평가자 개인 페이지]
            │
            ├── 시험 결과 요약 확인
            │   ├── 총점, 과목별 점수
            │   ├── 획득 역량 수
            │   ├── 대비 통계
            │   └── 총평
            │
            └── [평가 세부 버튼] 클릭
                    │
                [평가 세부 내용 페이지]
                    ├── 평가 요약 (점수, 역량수, 소요시간, 통계, 총평)
                    └── 역량별 상세 (획득여부/점수, 코멘트, 통계 그래프)
```

## 4. 화면 상세 기획

### 4.1 메인 페이지 (평가 개요 대시보드)

**페이지 목적:** 전체 평가 현황을 한눈에 파악하고 평가자 목록을 관리

---

#### 4.1.1 상단 영역 — 통계 카드

| 컴포넌트 | 표시 데이터 | 비고 |
|----------|------------|------|
| 전체 응시자 수 카드 | N명 | 전체 대상자 대비 표시 |
| 응시 완료 카드 | N명 / 완료율 % | 완료/미완료 비율 포함 |
| 평균 총점 카드 | X점 | 최고점 / 최저점 병기 |
| 평균 소요시간 카드 | HH:MM | 제한시간 대비 표시 |
| 역량 평균 획득 수 카드 | N개 / 전체 M개 | 선택적 추가 항목 |

---

#### 4.1.2 상단 영역 — 시각화 차트

| 컴포넌트 | 차트 유형 | 표시 데이터 | 비고 |
|----------|----------|------------|------|
| 점수 분포 차트 | 막대 그래프 (히스토그램) | 점수 구간별 인원 수 | 구간: 0~20, 20~40, ... |
| 과목별 평균 점수 차트 | 레이더 차트 or 수평 막대 | 각 과목의 평균 점수 | 과목 수에 따라 유형 선택 |
| 응시 완료 비율 차트 | 도넛 차트 | 완료 / 미완료 비율 | 중앙에 완료율 % 표시 |
| 시간대별 응시 현황 | 선 그래프 | 날짜별 응시 인원 추이 | 선택적 추가 항목 |

---

#### 4.1.3 하단 영역 — 평가자 목록

**컴포넌트명:** `ExamineeTable`

**필터 & 정렬 컨트롤**

| 컨트롤 | 타입 | 옵션 |
|--------|------|------|
| 응시 상태 필터 | 탭 or 셀렉트 | 전체 / 완료 / 미완료 |
| 정렬 기준 | 셀렉트 | 총점수 / 소요시간 |
| 정렬 방향 | 토글 버튼 | 오름차순 / 내림차순 |
| 검색 | 텍스트 입력 | 이름, 이메일 검색 |

**테이블 컬럼**

| 컬럼명 | 데이터 | 정렬 가능 | 비고 |
|--------|--------|----------|------|
| 이름 | 문자열 | N | - |
| 이메일 | 문자열 | N | - |
| 응시 상태 | 배지 (완료/미완료) | N | 색상으로 구분 |
| 총점수 | 숫자 (점) | Y | - |
| 과목별 점수 | 숫자들 | N | 과목명 툴팁 or 컬럼 분리 |
| 소요시간 | HH:MM:SS | Y | - |
| 상세 보기 | 버튼/행 클릭 | - | 개인 페이지로 이동 |

**페이지네이션**

- 기본 20행/페이지
- 페이지당 행 수 변경 옵션: 10 / 20 / 50

---

### 4.2 평가자 개인 페이지

**페이지 목적:** 특정 평가자의 전체 시험 결과를 상세하게 파악

**진입 경로:** 메인 페이지 평가자 목록 행 클릭

---

#### 4.2.1 페이지 헤더

| 컴포넌트 | 내용 |
|----------|------|
| 뒤로가기 버튼 | 메인 페이지로 복귀 |
| 평가자 이름 | 대형 타이틀 표시 |
| 이메일 | 서브 텍스트 |
| 응시 상태 배지 | 완료 / 미완료 |
| 응시 일시 | YYYY.MM.DD HH:MM |

---

#### 4.2.2 상단 영역 — 시험 결과 요약

**Summary Card 그룹**

| 카드 | 데이터 | 추가 정보 |
|------|--------|----------|
| 총점 | X점 / 만점 | 합격/불합격 여부 (기준 있을 경우) |
| 획득 역량 수 | N개 / 전체 M개 | 역량 획득률 % |
| 총 소요시간 | HH:MM:SS | 제한시간 대비 % |
| 백분위 | 상위 N% | 전체 응시자 기준 |

**과목별 점수**

| 컴포넌트 | 유형 | 내용 |
|----------|------|------|
| 과목별 점수 바 | 수평 Progress Bar | 과목명, 획득점수, 만점, 비율 |
| 레이더 차트 | 차트 | 과목간 점수 분포 시각화 |

**대비 통계 (vs 전체 평균)**

| 항목 | 표시 방식 |
|------|----------|
| 총점 비교 | 본인 점수 vs 평균 점수, 차이(+/-) 표시 |
| 과목별 비교 | 바 차트로 나란히 비교 |
| 소요시간 비교 | 본인 vs 평균, 빠름/느림 표시 |
| 역량 획득 수 비교 | 본인 vs 평균 |

**시험 결과 총평**

| 컴포넌트 | 내용 |
|----------|------|
| 총평 텍스트 박스 | AI 생성 or 관리자 작성 총평 |
| 강점 태그 | 우수한 역량 영역 하이라이트 |
| 개선 필요 태그 | 부족한 역량 영역 하이라이트 |

> **추가 권장 데이터 (자체 판단)**
> - 응시 시작/종료 시각
> - 문항별 정오답 요약 (정답률)
> - 이전 응시 이력 비교 (재응시 경우)

---

#### 4.2.3 하단 영역 — 평가별 세부 내용 접근

| 컴포넌트 | 내용 |
|----------|------|
| 평가 카드 목록 | 각 평가(과목)마다 카드 형태로 나열 |
| 카드 내 정보 | 평가명, 점수, 역량 획득 수, 소요시간 |
| CTA 버튼 | "세부 내용 보기" → 평가 세부 페이지 이동 |

---

### 4.3 평가 세부 내용 페이지

**페이지 목적:** 특정 평가에 대한 역량별 상세 결과 확인

**진입 경로:** 평가자 개인 페이지 > 평가 카드 "세부 내용 보기" 클릭

---

#### 4.3.1 페이지 헤더

| 컴포넌트 | 내용 |
|----------|------|
| 브레드크럼 | 메인 > [평가자명] > [평가명] |
| 평가명 | 대형 타이틀 |
| 평가자 이름 | 서브 텍스트 |

---

#### 4.3.2 상단 영역 — 평가 요약

**Summary Card 그룹**

| 카드 | 데이터 |
|------|--------|
| 평가 점수 | X점 / 만점 |
| 역량 획득 수 | N개 / 전체 M개 |
| 소요시간 | HH:MM:SS |
| 전체 평균 대비 | 상위 N% or +/- X점 |

**통계 비교 차트**

| 컴포넌트 | 유형 | 내용 |
|----------|------|------|
| 점수 분포 내 위치 | 히스토그램 + 마커 | 전체 분포에서 본인 위치 표시 |
| 평균 대비 바 차트 | 그룹 바 차트 | 본인 vs 전체 평균 |

**평가 총평**

| 컴포넌트 | 내용 |
|----------|------|
| 총평 텍스트 | 해당 평가에 대한 종합 평가 |
| 역량 요약 태그 | 획득/미획득 역량 태그 목록 |

---

#### 4.3.3 하단 영역 — 역량별 상세

> **화면 구성 판단:** 역량 수가 많을 경우(5개 이상) 아코디언(토글) 방식 권장, 적을 경우 전체 펼침 방식 권장

**역량 아이템 (반복 컴포넌트)**

| 구성 요소 | 내용 | 비고 |
|----------|------|------|
| 역량명 | 텍스트 | 헤더 역할, 클릭 시 토글 |
| 획득 여부 배지 | 획득 / 미획득 | 색상 구분 (초록/빨강) |
| 점수 | X점 / 만점 Y점 | - |
| 코멘트 | 텍스트 | 해당 역량에 대한 피드백 |
| 통계 그래프 | 바 차트 | 전체 응시자 중 획득 비율, 본인 점수 위치 |

**전체 역량 요약 차트 (섹션 상단)**

| 컴포넌트 | 유형 | 내용 |
|----------|------|------|
| 역량 획득 현황 | 도넛 차트 | 획득 / 미획득 역량 비율 |
| 역량별 점수 비교 | 수평 바 차트 | 전체 역량을 한눈에 비교 |

## 5. 공통 컴포넌트 명세

| 컴포넌트명 | 설명 | 사용 화면 |
|-----------|------|----------|
| `StatCard` | 단일 수치 강조 카드 (라벨 + 값 + 서브텍스트) | 전 페이지 |
| `StatusBadge` | 상태 표시 배지 (완료/미완료/획득/미획득) | 전 페이지 |
| `ProgressBar` | 점수/비율 시각화 바 | 개인, 세부 페이지 |
| `RadarChart` | 과목별 점수 레이더 차트 | 개인 페이지 |
| `BarChart` | 단일/그룹 막대 차트 | 전 페이지 |
| `DonutChart` | 비율 도넛 차트 | 메인, 세부 페이지 |
| `LineChart` | 추이 선 그래프 | 메인 페이지 |
| `DataTable` | 필터/정렬 가능한 테이블 | 메인 페이지 |
| `Breadcrumb` | 페이지 경로 표시 | 세부 페이지 |
| `Accordion` | 역량별 상세 토글 | 세부 페이지 |
| `CommentBox` | 텍스트 총평/코멘트 박스 | 개인, 세부 페이지 |
| `TagGroup` | 역량/강점/약점 태그 그룹 | 개인, 세부 페이지 |
